Description du projet

Développement complet d’une application web dans le cadre de mon Projet de Fin d’Études (PFE) chez Itqan Labs. Le projet repose sur la stack MERN (MongoDB, Express.js, React.js, Node.js) et vise à fournir une solution de gestion digitale pour les établissements éducatifs.

Fonctionnalités principales : gestion des utilisateurs avec rôles, tableau de bord avec visualisation des statistiques, sécurité via authentification JWT, design responsive et intégration d’API externes. Déploiement réalisé sur GitHub Pages.

Fonctionnalités clés

Responsive Design

Interface adaptative compatible sur tous les supports (desktop, mobile, tablette).

Authentification sécurisée

Système d’authentification via JWT, gestion sécurisée des sessions et hachage des mots de passe.

Statistiques en temps réel

Visualisation des données avec des graphiques dynamiques, filtres personnalisés.

API RESTful

Architecture backend modulaire avec Express.js pour une communication efficace via API REST.

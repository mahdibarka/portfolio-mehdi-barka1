Aperçu du projet

Développement d’une application web dans le cadre de mon projet de fin d’études chez Itqan Labs. Stack utilisée : React.js, Node.js, MongoDB, Express.js.

Fonctionnalités : gestion d’utilisateurs, tableaux de bord avec statistiques, authentification sécurisée, intégration API et design responsive.

Fonctionnalités clés

Design Responsive

Compatible avec tous les écrans grâce à Bootstrap 5.3.

Sécurité

Authentification sécurisée, gestion des sessions, protection des routes.

Statistiques dynamiques

Dashboards avec visualisation des données en temps réel.

Architecture RESTful

Backend structuré avec Express.js et gestion via API REST.

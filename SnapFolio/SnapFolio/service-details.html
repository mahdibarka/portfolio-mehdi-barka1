Description du projet

Développement complet d’une application web dans le cadre de mon PFE chez Itqan Labs. Stack : React.js, Node.js, MongoDB, Express.js. Hébergement sur GitHub Pages.

Fonctionnalités : gestion des utilisateurs, authentification JWT, visualisation des statistiques dynamiques, intégration d’API, UI responsive.

Fonctionnalités clés

Responsive Design

Interface adaptative sur tous les supports (PC, mobile, tablette).

Authentification sécurisée

Connexion via JWT, sessions protégées, hashing des mots de passe.

Statistiques dynamiques

Dashboards en temps réel avec filtres et graphiques interactifs.

Architecture REST

Backend Express.js organisé en API RESTful modulaire.
